# CodeAlpha — Build a Calculator

A simple browser-based calculator built with HTML, CSS, and JavaScript.

## Overview
This repository contains a lightweight calculator web app that performs basic arithmetic operations (addition, subtraction, multiplication, division) and includes a clean, responsive UI.

## Features
- Basic arithmetic: +, -, ×, ÷
- Clear and delete functions
- Responsive layout for desktop and mobile

## Development
- Edit `script.js` to change logic.
- Edit `style.css` to update appearance.
- No build step required.
