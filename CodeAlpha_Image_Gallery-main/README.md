# CodeAlpha Image Gallery

A lightweight frontend image gallery built with vanilla HTML, CSS, and JavaScript.

## Overview

This project demonstrates a responsive image gallery with category filtering and a lightbox viewer.

## Features

- Responsive grid-based gallery layout
- Category filter buttons for `Nature`, `Architecture`, and `Animals`
- Clickable images that open in a lightbox modal
- Next / previous navigation inside the lightbox
- Hover overlay and zoom effect for gallery items

## Files

- `image_gallery.html` — main HTML page for the gallery
- `style.css` — styling for layout, filters, gallery cards, and lightbox
- `script.js` — gallery filtering logic and lightbox controls

## Usage

1. Open `image_gallery.html` in a browser.
2. Click filter buttons to show images by category.
3. Click an image to open the lightbox viewer.
4. Use the left/right arrows to navigate or click the close button to exit.

## Notes

- Images are loaded from remote URLs using `picsum.photos`.
- No build or package manager is required.
- Works in modern browsers with JavaScript enabled.
